node-gitlab
===========

[GitLab](https://github.com/gitlabhq/gitlabhq) API Nodejs library.

Install
=======

```bash
npm install gitlab
```

Develop
=======

```bash
cake watch
```

Edit the [CoffeeScript](http://coffeescript.org/) files in `src`.

Example Usage
========

See [Examples](https://github.com/moul/node-gitlab/tree/master/examples)
